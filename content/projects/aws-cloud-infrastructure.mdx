# AWS Cloud Infrastructure Project

A comprehensive cloud infrastructure solution that demonstrates enterprise-level deployment practices with automated CI/CD pipelines and secure data management.

## Project Overview

This project showcases the deployment of a robust Windows-based web application on AWS infrastructure, featuring automated deployment pipelines, secure authentication, and real-time data synchronization from Active Directory.

## Key Features

- **Cloud-Native Deployment**: Windows-based web server hosted on AWS
- **CI/CD Pipeline**: Automated deployment using CDK, Bitbucket, and Bamboo
- **Secure Authentication**: Windows Authentication with strict access controls
- **Automated Data Sync**: Real-time Active Directory integration
- **Dynamic Web Interface**: Data fetching and display from multiple sources

## Technical Architecture

### Cloud Infrastructure (AWS)

- **EC2 Windows Server**: Hosted web application server
- **CDK Pipeline**: Infrastructure as Code deployment
- **VPC Configuration**: Secure network architecture
- **Security Groups**: Controlled access and traffic management

### CI/CD Pipeline

- **Bitbucket**: Source code management and version control
- **Bamboo**: Continuous integration and deployment automation
- **Automated Testing**: Quality assurance in deployment pipeline
- **Environment Management**: Staging and production deployments

### Data Management

- **SQL Database**: Centralized data storage and management
- **Active Directory Integration**: Automated user and organizational data sync
- **PowerShell Automation**: Scheduled tasks for data population
- **ADSI Access**: Direct Active Directory Service Interface integration
